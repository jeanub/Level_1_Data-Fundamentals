# Nivel 1 - Fundamentos del Dato

**Ruta de Aprendizaje en Business Intelligence y Extracción de Datos - DataNova Corp**

---

## Objetivo General
Construir el pipeline base de Business Intelligence:  
**Extracción → Limpieza → Exploración → Visualización → Automatización**  
Este nivel establece las bases de la arquitectura analítica de la empresa.

---

## Competencias Desarrolladas
- Extracción de datos desde archivos y web  
- Limpieza y normalización con **Pandas**  
- Análisis exploratorio (EDA) y correlaciones  
- Creación de dashboards en **Power BI**  
- Automatización de reportes (**Python / Power Automate**)  
- Control de versiones con **Git**  
- Documentación técnica y analítica  

---

## Proyectos Realizados

### 1) Transformación de dataset inicial
**Objetivo:** convertir un dataset caótico en información limpia y confiable  
**Logros:**  
- Eliminación de nulos y duplicados  
- Estandarización de columnas y tipos de datos  
- Validaciones automáticas de calidad  
**Competencias:** limpieza, estructuración, validación  

---

### 2) Dashboard de ventas
**Objetivo:** construir un tablero interactivo con KPIs básicos  
**Logros:**  
- KPIs de ingresos, pedidos y ticket promedio  
- Filtros por año, región y producto  
- Visualización orientada a toma de decisiones  
**Competencias:** visualización, diseño, storytelling analítico  

---

### 3) Análisis exploratorio de dataset
**Objetivo:** identificar relaciones entre variables y patrones relevantes  
**Logros:**  
- Correlaciones visuales (heatmaps, scatterplots)  
- Detección de outliers y variables críticas  
- Síntesis de insights con impacto  
**Competencias:** análisis exploratorio, pensamiento crítico, interpretación  

---

## Notas
Este nivel sienta las bases técnicas para arquitecturas analíticas más avanzadas, priorizando reproducibilidad, calidad y escalabilidad.
